# Kanban Board (React + Vite + Tailwind)

A simple Kanban Board application built using React, Vite, React Router and Tailwind CSS.

---

##  Features

- Login System (username stored in localStorage)
- Add Tasks
- Move Tasks (Todo → In Progress → Done)
- Delete Tasks
- Data persists after refresh (localStorage)
- Modern UI using Tailwind CSS
- Responsive layout

---

##  Tech Stack

- React
- Vite
- React Router DOM
- Tailwind CSS
- LocalStorage (for data persistence)

1 Clone the repository


2 Install dependencies


3️ Run the development server


4️ Open in browser


---

## How It Works

- User logs in with a username.
- Tasks are added to the Todo column.
- Tasks can be moved to In Progress and Done.
- All tasks are stored in localStorage.
- Data remains after refresh.
